Responsive CSS Cheat Sheet

Quick reference for Filters, Media Queries, Typography, and Responsive Design

Filters

filter: grayscale(100%); — apply visual effects to images.

Media Queries

@media (max-width: 700px) { ... } — change layout per viewport.

Responsive Design

Use flex, grid, and relative units (%, vw, rem).

Typography

font-size: clamp(1rem, 2.5vw, 1.25rem); — fluid readable text.

Images

max-width:100%; height:auto; — make images responsive.

Spacing

Use gap, padding, and logical margins for rhythm.
