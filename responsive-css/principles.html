Responsive Design Principles

Fluid Layouts

Layouts that resize using relative units (%, vw, fr).

grid-template-columns: 1fr 2fr;

Flexible Media

Images/media scale with container (max-width:100%; height:auto).

img { max-width:100%; height:auto; }

Breakpoints

Use media queries for layout changes at targeted widths.

@media (max-width: 600px) { ... }
